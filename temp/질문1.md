- jQuery.get(url [, data ] [, success ] [, dataType ] )

  - 메서드가 위와 같은 시그니처를 갖고 있는데, (1번 String, 2번 PlainObject 또는 String)

  - ```javascript
    $.get( "ajax/test.html", function( data ) {
      $( ".result" ).html( data );
      alert( "Load was performed." );
    });
    ```

    - 1번 arg: string
    - 2번 arg: function
